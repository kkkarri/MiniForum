\# 🌐 Вебплатформа «MiniForum»



\[cite\_start]\*\*MiniForum\*\* — це легкова інтерактивна вебплатформа на базі архітектурного шаблону \*\*ASP.NET Core MVC\*\*, розроблена в межах навчальної практики з комп'ютерного програмування\[cite: 110, 118]. \[cite\_start]Застосунок призначений для швидкого обміну інформацією та координації всередині студентських спільнот\[cite: 122].



\---



\## 🛠️ Технологічний стек

\* \[cite\_start]\*\*Фреймворк:\*\* ASP.NET Core 8 (MVC) \[cite: 118, 144]

\* \[cite\_start]\*\*База даних:\*\* SQLite \[cite: 118, 145]

\* \[cite\_start]\*\*ORM:\*\* Entity Framework Core (Підхід Code First + Автоматичне розгортання через `Database.EnsureCreated()`) \[cite: 118, 145]

\* \[cite\_start]\*\*Фронтенд:\*\* Razor Pages (HTML5, CSS3, Bootstrap 5, JavaScript Fetch API для AJAX-запитів) \[cite: 132, 144]



\---



\## 🚀 Основний функціонал



\### 👤 Модуль користувача

1\. \[cite\_start]\*\*Головна сторінка:\*\* відображає динамічний список розділів («Навчання», «Ігри», «Флудилка») із підрахунком кількості тем у кожному з них за допомогою LINQ-запитів\[cite: 125, 147].

2\. \[cite\_start]\*\*Список тем:\*\* виводить назву, автора, дату створення та кількість відповідей з автоматичним сортуванням топіків за часом останньої активності\[cite: 126, 127, 128].

3\. \[cite\_start]\*\*Створення тем та обговорення:\*\* користувачі можуть створювати нові теми та залишати коментарі через форми з валідацією на порожні поля\[cite: 129, 130, 146].

4\. \[cite\_start]\*\*Повнотекстовий пошук:\*\* реалізовано фільтрацію даних одночасно за назвою теми та вмістом повідомлень\[cite: 131].

5\. \[cite\_start]\*\*Асинхронні лайки:\*\* оцінювання коментарів працює без перезавантаження сторінки через Fetch API\[cite: 132].



\### 👑 Модуль модератора

1\. \[cite\_start]\*\*CRUD розділів:\*\* адміністративна панель для додавання, редагування та видалення категорій форуму\[cite: 134].

2\. \[cite\_start]\*\*Керування станом тем:\*\* функція закриття/відкриття тем із миттєвим блокуванням форми відповідей для звичайних користувачів\[cite: 136, 146].

3\. \[cite\_start]\*\*Модерація контенту:\*\* можливість безповоротного видалення некоректних повідомлень\[cite: 136].

\## 🏃‍♂️ Як запустити проєкт локально



1\. Переконайся, що у тебе встановлено \*\*.NET 8 SDK\*\*.

2\. \[cite\_start]Відкрий рішення `MiniForum.sln` у \*\*Visual Studio 2022\*\*\[cite: 150].

3\. Переконайся, що в конструкторі `ForumDbContext` викликається метод `Database.EnsureCreated()`.

4\. Натисни кнопку \*\*`F5`\*\* або `Start Debugging`. 

5\. \[cite\_start]Проєкт автоматично згенерує локальну базу даних `forum.db`, наріже таблиці та заповнить їх демонстраційними даними\[cite: 145]. Застосунок відкриється у браузері.
